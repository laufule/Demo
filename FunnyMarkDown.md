## 表格 ##

表格的一种写法：

|name|age|score|
|:--|:--:|:--:|
|name1|18|93|
|name2|10|95|
|name3|13|89|

表格的另一种写法：

one|two|therr|four
:--:|:--:|:--:|:--:
1|2|3|4
4|5|6|7
9|43|2|1

第一种看着舒服些，第二种写法简单些。

## 代码 和 纯文本 ##

实现代码高亮，方法如下：

```C++
#include <iostream>
int main()
{
    int age = 17;
    std::cout << "Your age is " << age << endl;
    return 0;
}
```

纯文本首行缩进，默认为代码或者纯文本，如：

这里的文字就不会被识别为MarkDown格式了
